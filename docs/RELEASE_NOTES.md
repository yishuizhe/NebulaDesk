# NebulaDesk 0.1.0

Initial public preview.

- Local prompt-to-wallpaper renderer
- Animated nebula canvas with starfield, glow rings, particles, and grain
- Prompt presets and random scene generator
- Intensity, velocity, gravity, and grain controls
- PNG export
- Preset save/load
- Shareable scene seed copy
- macOS desktop wallpaper window prototype
- macOS DMG/ZIP and Windows NSIS/portable build scripts
