# Product Notes

NebulaDesk is designed as a "small but viral" desktop creative tool.

## Hook

The user does not pick from a wallpaper catalog. They type a vibe and get a living scene immediately.

## Viral loops

- Scenes can be described by short prompts.
- Scene seeds can be copied into social posts.
- PNG export creates shareable stills.
- Presets are plain JSON files that can be posted anywhere.

## Technical stance

- The renderer is deliberately local-only.
- The first version avoids paid AI APIs so the project remains easy to run.
- Electron is used because the wallpaper mode and packaging story matter more than a web demo.

## Next high-impact improvements

- Add Windows WorkerW wallpaper embedding.
- Add recording export for short looping videos.
- Add a preset gallery stored as static JSON.
- Add audio-reactive motion as an optional local mode.
